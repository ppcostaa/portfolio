Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro. Apresenta experiências profissionais, skills técnicas e formas de contato, com animações de scroll, menu responsivo e navegação suave.

---

## ✨ Funcionalidades

- Navegação com scroll suave e link ativo automático conforme o scroll
- Menu hamburguer para dispositivos móveis
- Animações de entrada (scroll-reveal) com `IntersectionObserver`
- Hover animado nos títulos de seção
- Layout totalmente responsivo (mobile, tablet e desktop)
- Tipografia com Playfair Display + DM Sans via Google Fonts

## 🚀 Como rodar localmente

**Opção 1 — abrir direto no navegador**
```bash
git clone https://github.com/ppcostaa/portfolio.git
cd portfolio
# abra o index.html no seu navegador
```

**Opção 2 — com Live Server (VS Code)**
- Instale a extensão **Live Server** no VS Code
- Clique com o botão direito em `index.html` → *Open with Live Server*
